# Patrón de Diseño Builder en Java

Este proyecto es un ejemplo práctico del **Patrón de Diseño Builder**, un patrón creacional que permite construir objetos complejos paso a paso, mejorando la legibilidad y organización del código.

 ¿Qué es el Patrón Builder?

El patrón **Builder** separa la construcción de un objeto de su representación final.  
Se utiliza cuando un objeto tiene muchos parámetros o requiere un proceso de construcción controlado.

  Ventajas

- Mejora la legibilidad del código
- Facilita la creación de objetos complejos
- Evita constructores largos y difíciles de leer
- Hace posible diferentes configuraciones del mismo objeto

Tecnologías

- Java 17 (o versión compatible)
- Visual Studio Code / PowerShell
